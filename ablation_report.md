## 核心发现

1. **金融板块 NLP 完全无效**：5 支金融股全部 NLP 负贡献，可能是金融新闻噪音多、信号弱
2. **消费板块 NLP 效果最好**：80% 正向率，消费品新闻情感与股价关联更直接
3. **NFLX 是最强受益者**：NLP 贡献了 +0.97 Sharpe 和 +525% 额外收益
4. **NVDA 受 NLP 影响最严重**：Sharpe 从 0.65 跌至 -0.76（Δ -1.41）
5. **整体结论**：当前 NLP 情感信号对 DQN 策略的帮助有限（39.3% 正向率），不构成系统性优势，但特定板块（消费、通信）可受益

## 优化后重跑：金融板块 + NVDA（2026-05-08）

**新增优化**（相比基线实验新增 7 项）：

| # | 优化 | 说明 |
|---|---|---|
| 1 | 共识加权 | 方法间相关性 < 0.6 时，FinBERT/LLM 权重 2x，LR 1.5x |
| 2 | EMA 平滑 | span=5，平滑日间噪声 |
| 3 | 中性门控 | `|score| < 0.1` → 清零，过滤弱信号 |
| 4 | 奖励塑形 | 情感-仓位对齐时 ±0.0002 奖励 |

| 7 | Double DQN + LR decay | 双网络解耦 + ExponentialLR(0.995) + 加载最佳 checkpoint |

### DQN 训练方差测量

同一条件（with-NLP）不同 run 之间的 Sharpe 波动：

| 股票 | Run 1 Sharpe | Run 2 Sharpe | 波动幅度 |
|------|-------------|-------------|---------|
| V | -0.62 | -0.03 | **0.59** |
| GS | 0.78 | 1.02 | **0.24** |
| JPM | 0.56 | 0.56 | 0.00 |

V 的两次训练结果相差 0.59 Sharpe，远超情感信号的边际贡献。**DQN 训练方差本身淹没了弱情感信号**。

### 关键发现

1. **NVDA 翻正（确认有效）**：Sharpe Δ 从 -1.41 → +0.13，提升 +1.54。NVDA 情感波动大、信号强，优化有效。
2. **金融板块：NLP 无效（确认）**：即使降低门控阈值、增加情感天数，DQN 仍无法从金融情感中获益。金融情感信号（均值 0.03~0.08）太弱，被价格特征淹没。
3. **DQN 方差 > 弱情感信号**：同条件下 V 的 Sharpe 波动 0.59，远大于 NLP 的边际贡献（±0.1~0.3）。对弱信号股票，单次训练结果不可靠。
4. **MA 是唯一例外**：两次 run 均显示 NLP 负贡献（-0.49, -0.47），说明其情感信号有系统性误导。

### 结论

- **NLP 情感信号对 DQN 策略的整体帮助有限**，仅在情感波动大的股票（NVDA）上有正向贡献
- 金融板块（5 支）NLP 总体无效或轻微有害
- 进一步优化建议：多种子训练（≥3 seeds）量化方差，或对弱信号股票直接关闭 NLP 输入

### 核心发现

1. **真实新闻改变了答案**：模板数据下金融板块 NLP 有效率 0%（0/5），真实数据下 40%（2/5：JPM + V）。之前"金融 NLP 无效"的结论是基于假数据的误判。

2. **NVDA 持续正向（+0.20）**：模板优化版 +0.13，真实版 +0.20。NVDA 是唯一在两种数据下都确认 NLP 有效的股票。

3. **WMT 是最大受益者（+0.32）**：Sharpe 从 0.66 → 0.98，Return 从 +12.6% → +31.7%。消费品新闻情感与股价的关联更直接。

4. **部分股票仍无差异**（BAC、MA、KO）：尽管用的是真实新闻，with/without 结果仍然完全一致（到 4 位小数）。需要排查是门控清零还是 DQN 收敛问题。

5. **真实新闻情感质量明显更高**：Fleiss' Kappa 显示方法间有实质分歧（-0.57 ~ +0.38），不像模板数据的虚假一致。这说明真实新闻的语义是多样化的。

### 最终结论

- **真实新闻显著提升了 NLP 情感信号的价值**：NLP 有效率从模板数据的 ~20% 提升到 50%（4/8）
- **金融板块并非根本上不适合 NLP**：2/5 正向，问题在于之前的模板数据无法产生有意义的情感信号
- **DQN 收敛仍有波动**：BAC/MA/KO 的零差异需要进一步诊断（多种子训练或降低门控阈值）
- 建议：继续用 Finnhub 采集新闻，积累更长历史数据后做完整的 28 股消融
