## 按板块交叉分析

| 板块 | NLP 正向率 | 正向股票 | 结论 |
|---|---|---|---|
| 能源/工业 | **80%** (4/5) | XOM, CVX, CAT, NEE | NLP 对周期性行业最有效 |
| 消费 | 40% (2/5) | PEP, + WMT 严重拖累 | 板块内分化严重 |
| 通信 | 50% (1/2) | NFLX | DIS 负向 |
| 医疗 | 25% (1/4) | PFE | UNH/JNJ 大幅拖累 |
| 科技 | 29% (2/7) | AAPL, TSLA | GOOGL/NVDA 负向出乎意料 |
| 金融 | **0%** (0/5) | — | NLP 对金融板块完全无效 |

---

## 核心发现

1. **NLP 对周期性行业最有效**：能源（XOM, CVX）、工业（CAT）、公用（NEE）均正向，情感信号与价格动量共振强
2. **金融板块 NLP 完全失败**：5 支金融股中 0 支正向，经济数据驱动的金融新闻不适合情感分析
3. **情感信号稀疏度是核心瓶颈**：7 支股票（25%）Shapre Δ = 0，说明 DQN 完全忽略了情感信号——每日情感覆盖不足
4. **LLM 第 4 方法加入后并未显著改善**：相比上一轮（39.3% 正向），本轮仅 32.1%，可能因为 LLM 仅覆盖 10 篇文章/股票（受限于 API 速率）
5. **PEP 表现异常突出**：Sharpe Δ = +2.49，消费必需品板块中情感信号质量最高
